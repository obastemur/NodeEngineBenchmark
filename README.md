Run sunspider tests locally on [JXcore](https://github.com/jxcore/jxcore) to measure the embedded JavaScript engine performance.

#### How To

```
jx runTest.js
```

MacBook Pro i7-2.7Ghz (JXcore 0.3.0.1 SM 34 / V8 3.14) Results;

**V8**: 330.6 ms
